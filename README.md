README (Português)

LoL Match Monitor

LoL Match Monitor é uma aplicação criada especialmente para streamers de League of Legends e equipes de suporte técnico que desejam monitorar estatísticas detalhadas de partidas em tempo real.

O objetivo é oferecer uma visão mais completa do desempenho do jogador, exibindo informações que normalmente não aparecem na tela durante a transmissão, mas que são úteis para análise, feedback e melhoria da gameplay.

Funcionalidades

Monitoramento em tempo real: coleta de dados diretamente da API do LoL durante a partida.

Estatísticas detalhadas: KDA, CS/min, dano causado, gold, visão, rota e mais.



README (English)
 LoL Match Monitor
LoL Match Monitor is an application designed specifically for League of Legends streamers and technical support teams who want to monitor detailed match statistics in real time.

Its goal is to provide a deeper view of player performance, displaying information that usually isn’t visible on stream, but is extremely useful for analysis, feedback, and gameplay improvement.

Features:

Real-time monitoring: pulls data directly from the Riot API during the match.

Detailed stats: KDA, CS/min, damage dealt, gold earned, vision score, lane data, and more.
